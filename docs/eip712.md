# EIP-712 Domain & Example

EIP-712 Domain:
- name: "MantlePersona"
- version: "1"
- chainId: variable (use Mantle chain id in production)
- verifyingContract: PersonaRegistry contract address

EIP-712 typed data example (JSON):

{
  "types": {
    "EIP712Domain": [
      {"name": "name", "type": "string"},
      {"name": "version", "type": "string"},
      {"name": "chainId", "type": "uint256"},
      {"name": "verifyingContract", "type": "address"}
    ],
    "Persona": [
      {"name": "personaId", "type": "bytes32"},
      {"name": "metadataCID", "type": "string"},
      {"name": "timestamp", "type": "uint256"},
      {"name": "nonce", "type": "uint256"}
    ]
  },
  "primaryType": "Persona",
  "domain": {"name": "MantlePersona", "version": "1", "chainId": 1, "verifyingContract": "0x..."},
  "message": {"personaId": "0x...", "metadataCID": "ipfs://...", "timestamp": 1650000000, "nonce": 1}
}
